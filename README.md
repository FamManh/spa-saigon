# Spa Manager
Ứng dụng tính toán danh thu của nhân viên spa theo ngày, tháng.

## Những công cụ đã sử dụng:
* Front-end: Reactjs + Redux + Redux-thunk + antd
* Back-end: Nodejs + Expressjs
* CSDL: Mongodb
